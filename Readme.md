# Sheriyans Coding School Website Clone

A responsive website clone of Sheriyans Coding School's landing page, built as a practice project to revise HTML and CSS fundamentals.

## Project Goal

The main objective of this project is to:
- Practice and revise HTML & CSS basics
- Master responsive web design principles
- Create a fully responsive website that works across all device sizes
- Implement modern CSS techniques like Flexbox and media queries

## Technologies Used

- HTML5
- CSS3
- Remix Icons (for icons)
- Custom fonts (Helvetica Now Display, Neue Machina)

## Features

- Fully responsive navigation bar
- Hamburger menu for mobile devices
- Flexible layout using Flexbox
- Custom CSS variables for consistent theming
- Responsive typography and spacing
- Mobile-first design approach

## Responsive Breakpoints

- Desktop: Full navigation menu
- Tablet/Mobile (< 600px): Adjusted padding and layout
- Mobile (< 500px): Hamburger menu enabled

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Resize the browser window to see responsive behavior

## Learning Outcomes

- Understanding of CSS Flexbox
- Media queries for responsive design
- CSS custom properties (variables)
- Layout positioning and spacing
- Mobile-first development approach
